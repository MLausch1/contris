# Introduction #

This Page shows the Controls of the downloadable .exe File


# Details #


Controls:
  * →   Move Block Right
  * ←   Move Block Left
  * ↑   Smash Bloock down
  * ↓   Push block down 1 Unit
  * S   Turn Clockwise
  * A   Turn Clockwise 3 Times ;-)
  * G   Turn ghost Block on and off (Default on)
  * N   Turn Next Block on and off
  * ESC Exit Game
